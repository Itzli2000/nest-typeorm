# Teslo API

1. Clone the repository

```bash
git clone https://github.com/teslo-shop/teslo-api.git
```

2. Clone env file and rename it to .env

```bash
cp .env.example .env
```

3. Change/add env variables

4. Install dependencies

```bash
yarn install
```

5. Run Project DB

```bash
docker-compose up -d
```

6. Run the seed

```bash
http://localhost:3000/api/seed
```

7. Run the project

```bash
yarn start:dev
```
